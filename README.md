### [CONVERSA AI CHAT BOT](https://telegram.me/conversaaibot)

### Features 
- **Auto Ai Reply**
- **Image Scan Feature**
- **Customized Prompt**
- **Customized UI**
- **Normal Or pin broadcast**

### Deploy Start Command 
```
 gunicorn app:app & python3 main.py
```
### Deploy Build Command 
```
pip install -r requirements.txt
```
